Security model

⚠️

Password loss = data loss

There is no recovery by design. The encryption key is derived from your password; without it, the datasource cannot be decrypted — no backdoor, no reset. Keep your password somewhere safe before you create a datasource.

First-launch: unsigned build

Kouign Assistant is currently distributed as an unsigned build (no Apple Developer certificate, no Windows Authenticode). Your OS will warn you on first launch — this is expected. Follow the steps below to allow it.

macOS — Gatekeeper

  1. Open Finder and locate the Kouign Assistant.app
  2. Right-click (or Control-click) the app → Open
  3. Click Open in the Gatekeeper dialog

Alternatively: System Settings → Privacy & Security → scroll down → "Open Anyway" next to the Kouign Assistant entry.

Windows — SmartScreen

  1. Run the installer; SmartScreen will show a blue warning
  2. Click More info
  3. Click Run anyway

Code signing and notarization are planned for a future release.
